
# E-Commerce Sales Dashboard

## Project Overview

This project analyzes e-commerce sales data using MySQL and Power BI to generate business insights and interactive visualizations.

## Tools Used

* MySQL
* Power BI
* SQL
* CSV Dataset

## Dashboard Features

* Total Revenue KPI
* Total Orders KPI
* Monthly Order Trend
* Orders by City
* Top Products by Revenue

## Key Insights

* Identified top-performing cities based on order volume
* Analyzed monthly sales trends
* Found highest revenue-generating products

## Workflow

CSV Dataset → MySQL Database → Power BI Dashboard

## Files Included

* Power BI Dashboard (.pbix)
* Dashboard Screenshot
* SQL-based Data Modeling
